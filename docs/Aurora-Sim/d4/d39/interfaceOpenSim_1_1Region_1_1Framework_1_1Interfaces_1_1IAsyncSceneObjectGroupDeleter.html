Deletes the given groups to the given user's inventory in the given folderID.

Parameters:
actionThe reason these objects are being sent to inventory
folderIDThe folder the objects will be added into, if you want the default folder, set this to UUID.Zero
objectGroupsThe groups to send to inventory
AgentIdThe agent who is deleting the given groups (not the owner of the objects necessarily)
permissionToDeleteIf true, the objects will be deleted from the sim as well
permissionToTakeIf true, the objects will be added to the user's inventory as well
