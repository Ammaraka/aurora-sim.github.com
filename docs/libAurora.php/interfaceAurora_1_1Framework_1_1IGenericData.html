Parameters:
array$wantedValuean array of fields or operations to return. Must contain only strings.
string$tablethe name of the table to perform the query on.
object$queryFilteran instance of Aurora::Framework::QueryFilter
array$sortan array with strings for field names/operations for keys and booleans for values- TRUE to sort in ascending order, FALSE to sort in descending order.
mixed$startif NULL, the query expects to return all results from result zero. If $count is NULL and $start is an integer, will return $start results
mixed$countif NULL and $start is an integer, will return $start results. If $start and $count are integers, will return $count results from point $start
Returns:
array A one-dimensional array of all fields in the result rows.
