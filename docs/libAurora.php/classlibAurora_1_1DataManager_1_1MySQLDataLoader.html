Connect to database. We're deviating from the c# design here, using the constructor to connect to the database rather than a method that can be used anywhere during runtime.

Important note! Unlike the equivalent c#, this class does not create databases- the database must be created beforehand by the user (except in the case of SQLite)

Parameters:

string$migratorNamemigrator module
boolean$validateTablesspecifying TRUE must attempt to validate tables after connecting to the database
boolean$forceBreakingChangesTRUE forces breaking changes to be applied
string$mysqlVersionSpecifies the version of the MySQL server so queries can take into account the differences between query results
